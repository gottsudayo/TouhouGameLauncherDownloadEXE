3.配布バージョンについて
ここで配布しているバージョンはver2.0.0以降（2.0.1,2.0.2を除く）です。
それ以外のバージョンは配布しておりませんのでご注意ください。


4.結局何が言いたいのか
このリポジトリを準備したのはGoogleDriveさんが不信感を与える為です。
なので、避難場所としてこのリポジトリを用意しました。

ソースコードには何も問題はないのに、GoogleDriveさんは私のプログラムを
「ウイルス」とか「不審なファイル」として認識しているので、
私は悲しくなってしまいます。

まあ、私はソースコードに記載されているコード以外は配布しておりませんので、
そこはご安心下さい。

どうしても気になるなら、自分でpythonの環境を構築して私のソースコードをコピーして使ってもいいです。
但し、ソースコードやファイルの再配布は禁止です。


5.おわりに

ダウンロードしても全く問題はありません。
